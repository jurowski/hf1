Why doesn't this look right in Internet Explorer (IE)?

Please consider using Firefox or Chrome! This site works best with browsers that conform to web standards. Internet Explorer (IE) starting with version 9 attempts to finally do that, but it depends whether the browser is set to display pages in "quirks" vs. "compatibility mode". While we are working to develop workarounds for the 12% of our users who come to us with IE, as all web developers know, it is a huge undertaking with an operation as small as ours, so please pardon the dust in the meantime, or come back via Firefox, Chrome, Safari, or another standards-based browser!" --Sorry for that ... Sandon Jurowski (support@habitforge.com)
